ThreePresenter

Framework-agnostic 3D viewer library built on Three.js

Multiple formats (GLB, PLY, OBJ, NXS) Annotation system Built-in UI controls Easy to use

Zero Configuration

Just include the script and start loading 3D models. Works out of the box with sensible defaults.

Multi-Format Support

Load GLB, GLTF, PLY, OBJ, and Nexus multiresolution models seamlessly.

Annotations

Add point, line, and area annotations with interactive selection and picking.

Beautiful UI

Built-in Bootstrap-styled controls for camera, lighting, and screenshots.

Framework Agnostic

Use with React, Vue, Svelte, or vanilla JavaScript. Your choice!

TypeScript

Full TypeScript support with comprehensive type definitions.

Interactive Controls

Left Drag: Rotate camera
Right Drag: Pan view
Scroll: Zoom in/out
Home Button: Reset camera
Light Button: Toggle lighting
Camera Button: Switch perspective/ortho
